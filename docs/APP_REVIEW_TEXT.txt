TikTok Desktop Publisher is a reusable desktop tool for creators. Login Kit lets each user authorize their own TikTok account through OAuth with PKCE. user.info.basic identifies the authorized account in the interface. Content Posting API and video.publish query current creator settings, let the user choose an allowed privacy level and interaction controls, and publish a locally selected video after explicit consent. The app displays upload and processing status. Future dates are kept in a local queue and submitted only when due because TikTok Direct Post has no native scheduling. Tokens are stored locally in the OS keyring when available and are never included in source code or logs. Users can revoke access from the interface. The demo uses the Developer Portal sandbox and shows login, consent, creator info, video selection, privacy controls, upload, status, and the resulting private post.
