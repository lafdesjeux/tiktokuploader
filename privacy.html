Privacy Policy

Last updated: July 2026

TikTok Desktop Publisher is a local desktop application that creators can install and reuse. It does not operate a central user-account service, advertising system, analytics service, or remote database.

Data processed

After a creator chooses to connect TikTok, the application processes OAuth tokens, TikTok creator information, the selected local video, caption, privacy choice, interaction settings, commercial-content declarations, AI-content declaration, and publishing status.

Purpose

This information is processed only to authenticate the creator, display TikTok's current export options, upload content explicitly selected by the creator, and report the resulting status.

Storage

Videos remain on the creator's device until the creator authorizes an upload. OAuth secrets are stored in the operating-system keyring when available. A restricted local file may be used as a fallback on systems without a keyring. Scheduled jobs are stored in a local SQLite database.

Sharing

The application sends authorized account information and publishing data only to TikTok's official OAuth and Content Posting API endpoints. The project does not sell personal information or share it with advertisers or unrelated third parties.

Control and deletion

Creators can revoke TikTok authorization with the Disconnect function or through TikTok account settings. Local configuration and queue files can be deleted from the application's user configuration and data directories.

Open source

The source code is publicly inspectable. Security issues and privacy questions may be submitted through the repository's issue tracker.
